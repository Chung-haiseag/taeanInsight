# 태안 인사이트 — 제품 기획문서

> 지역 AI 인텔리전스 플랫폼. *무료 공공데이터 + AI로 "태안의 지금·다음·과거"를 한곳에서 보고 묻는다.*
> 최종 갱신: 2026-06-23 · 운영 상세는 [`RUNBOOK.md`](../RUNBOOK.md), 지역 이식은 [`docs/REGION_PORTING.md`](./REGION_PORTING.md).

---

## 1. 제품 개요

- **한 줄 정의**: 태안 지역의 실시간 현황·주간 전망·시민 저널리즘·시장 데이터를 AI로 묶은 지역 정보 플랫폼.
- **비전**: 소도시도 **저비용 AI 운영(월 ≤ 수십만 원)**으로 신뢰할 수 있는 지역 정보 서비스를 가질 수 있다.
- **차별점**:
  1. 모든 데이터를 **무료 공공 API**(기상청·에어코리아·국립해양조사원·국토부·오피넷·ITS·TourAPI)와 태안신문 아카이브로 구성.
  2. AI는 **근거 기반(RAG)·출처 표기·HITL 검수**를 기본으로 — 환각·무책임 발행 방지.
  3. **Cloudflare 단일 스택**(Workers·D1·R2·Workers AI)으로 인프라·비용 최소화.

## 2. 문제와 기회

- 지역 신문/지자체는 인력·예산 제약으로 **실시간 정보·데이터 가공·개인화**를 하기 어렵다.
- 주민·관광객·소상공인·기관은 흩어진 정보(날씨·바다·행사·시세·뉴스)를 **한곳에서, 내 맥락으로** 보고 싶어 한다.
- 공공데이터는 풍부하나 **결합·해석·전달**이 안 됨 → AI가 이 간극을 메운다.

## 3. 타깃·페르소나

| 페르소나 | 니즈 | 대응 화면 |
|---|---|---|
| **주민·관광객** | 지금 날씨·바다·행사, 옛 소식 | 지금 태안 · 태안뉴스 · AI 질의 |
| **소상공인(사장님)** | 우리 가게 오늘 무엇을 할까 | 사장님 홈(초개인화) |
| **B2B(협회·부동산·숙박체인·연구)** | 지역 시장 데이터·추세·다운로드 | B2B 대시보드 |
| **B2G(군청·산하기관)** | 정책 지표·부서 협업 | 부서 워크스페이스 · 대시보드 |
| **시민기자** | 쉬운 작성·AI 보조·정산 | 시민기자 에디터 |

## 4. 정보구조(IA) — "정본 1개 + 딥링크"

같은 데이터를 여러 화면이 다른 **질문(렌즈)**으로 다룬다. 중복은 재구현이 아니라 **교차링크**로 연결.

| 화면 | 한 역할 | 시간축 |
|---|---|---|
| **지금 태안(/live)** | 현재 상태 한눈에 | NOW |
| **태안뉴스(/news)** | 무슨 일이 있었나(사실 보도) | 과거 |
| **주간 리포트(/reports)** | 다음 주 전망·해석(서사) | 주간 |
| **B2B 대시보드(/dashboard)** | 장기 추세·다운로드(분석) | 기간 |
| **AI 질의(/query)** | 자연어로 묻고 근거로 답 | 즉시 |
| **내 페이지(/me)·사장님 홈** | 내 관심사로 필터·실행 | 개인 |
| **아카이브(/archive)** | 1990~ 기사 검색 | 역사 |
| **시민기자(/citizen)** | 참여 저널리즘 | — |

원칙: *데이터는 한 번만 만들고(metrics), 화면은 "지금/해석/추세/나"라는 서로 다른 질문에 답한다.*

## 5. 핵심 기능

### 5.1 지금 태안 (실시간 쇼케이스)
- 핵심지표 인포그래픽(수요·기온·대기질·자외선·수온·물때·아파트가·축제)
- 날씨·대기질, 바다(수온·파고·물때·서핑·해수욕지수·일출몰), 관광 수요지수, 유가
- **도로 실시간 CCTV**(ITS, 태안 국도 39곳, HLS)
- **해안 해무 스틸컷**(국립해양조사원)
- **"역대 오늘, 태안"**(같은 날짜 과거 주요뉴스 랜덤)

### 5.2 주간 인사이트 리포트 (HITL 발행)
- 매주 금 16:00 초안 자동 생성(Workers AI 5섹션) → **편집부 검토(HITL)** → 금 17:00 발행
- 섹션별 시각화(수요·날씨·바다·환경·부동산·축제) + 군정소식·카드뉴스·주간뉴스
- 관리자 발행 화면(미리보기·거버넌스 사전검사·발행/회수), 발행 시 구독자 Web Push
- 공개(로그인 불요), 관심사 기반 섹션 강조

### 5.3 AI 질의 (RAG)
- **LLM**: Workers AI `llama-3.3-70b`(무료). 근거 기반·출처[번호]·HITL 원칙.
- **근거 6종 자동 주입**(질문 키워드 감지):
  날씨·주말예보(단기+중기) / 바다(일출몰·물때·수온·파고) / 부동산(실거래 읍면·㎡단가·월별추이) / 관광수요 / 축제·**군청 주간행사 PDF 일정** / 아카이브(bm25)
- 출처 정제(공식 근거 항상 표시 + 인용 기사만), 근거 없으면 "못 찾음"
- 캐시(질문 정규화), 비용 기록(0원)

### 5.4 초개인화 — 사장님 홈
- 가게 프로필(업종·읍면) + 실데이터 기반 **수요·날씨·물때·상권**
- **규칙엔진**: 업종×수요×날씨×물때 → 우선순위 실행 제안(예: 고수요+맑음→"객실 풀가동", 큰 간조→"갯벌체험 안내")
- 온보딩에 가게 프로필 스텝, 팀(B2B)·부서(B2G) **공유 워크스페이스**(코드 가입·공유자료·메모)

### 5.5 시민기자 에디터
- 작성: 임시저장·미리보기·글자수·모바일
- **키워드 → AI 초안**(날조 방지 `[확인 필요]` 마커), **사진 업로드(R2)**
- **기사 CRUD/내 기사**, 실시간 PII·민감주제 점검, AI 보조(다듬기·요약·제목)
- **검수 루프**: 제출 → 관리자 승인(발행)/반려(사유) → 기자 화면 반영

### 5.6 B2B 대시보드 (지역 데이터 분석)
- KPI(수요·대기질·실거래·유가) + 시계열(미세먼지·기온·수요) + 기간 필터(주/월/분기) + **CSV 다운로드**
- 사장님 홈(개인 가게 행동)과 분리된 **지역 시장 분석** 정체성

### 5.7 아카이브
- 태안신문 1990~2025, 약 10.4만 건. FTS5 트라이그램 검색(+LIKE 폴백), 전체 건수·연도 범위 표시
- 디지털화(Vision OCR + Gemini 멀티모달), 관리자 검수

## 7. AI·거버넌스
- **HITL**: 리포트·시민기자 기사는 편집부 검수 후 발행.
- **AI 라벨**: human / ai_assisted / ai_generated, AI 콘텐츠는 출처 필수.
- **민감주제**(선거·범죄·의료·종교 등): 규칙 기반 분류 → AI 단독 차단·HITL 필수.
- **PII**: 자동 탐지·마스킹.
- **비용**: 운영 LLM은 Workers AI 무료. 월 한도·서킷브레이커(데모).

## 8. 기술 아키텍처
- **백엔드**: Cloudflare Workers + Hono (`taean-insight-api`)
- **프론트**: Next.js + OpenNext on Workers (`taean-insight`)
- **저장**: D1 `taean-archive`(기사·검색·캐시·선호도·기사·워크스페이스), R2 `taean-archive-photos`(지면·사진)
- **LLM**: Workers AI(운영) / Gemini(디지털화). 운영 Worker는 Claude API 미사용.
- **핵심 패턴**:
  - **로컬 크롤러 미러**: Worker가 못 닿는 소스(군청 해외IP 차단·ITS 9443포트)는 한국 IP 로컬 크롤러(launchd)가 받아 D1/R2로 적재.
  - **D1 캐시(SWR)**: workers.dev는 엣지캐시 불가 → 느린 외부 API는 D1에 저장·즉시 서빙·백그라운드 갱신·cron 워밍.
  - **지역 중앙화**(`region.ts`): 좌표·코드·관측소를 한 파일로 → 타 지역 이식 용이.

## 9. 운영·자동화 (cron / launchd)
- **Worker cron**: 자정 뉴스·환경 스냅샷·비용집계 / 30분 metrics·뉴스·해무 워밍 / 12시간 뉴스·군청목록 / 금16시 리포트 초안 / 아침 환경알림
- **launchd(로컬, 한국IP)**: 군청 6h(주간행사 PDF 포함) · 도로 CCTV 30분
- 새 기능은 RUNBOOK §5 기능 로그에 한 줄 기록.

## 10. 성능
- 태안뉴스 3.5s→0.85s, 해무 9.5s→0.7s, /live 3.0s→1.5s, metrics 콜드 9~16s→0.7s (D1 캐시·SWR·워밍·병렬화·엣지캐시)

## 11. 수익화·세그먼트
- **무료**: 지금 태안·뉴스·리포트·AI질의(일 5회)·아카이브
- **B2C Premium(사장님 Pro)**: 초개인화 홈·실행제안
- **B2B**: 지역 데이터 분석·CSV·API(향후)·AI질의(일 30회)
- **B2G**: 부서 워크스페이스·정책 지표
- 세그먼트별 한도(SEGMENT_LIMITS). 결제·entitlement는 데모(향후 실연동).

## 12. 로드맵

  초개인화 사장님 홈·워크스페이스, 시민기자(작성~검수 루프), B2B 대시보드, 성능·IA·모바일.
